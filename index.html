Functions

A function is a body of code that returns a value. a function can be made by declaring it and then giving it code to run. functions are usually used for putting together multiple methods so they can easily be recalled in the program. here is some raw code for creating a function.


            function whatTypeIn(){
                if (typeof userInput === 'string') {
                    console.log('yep, it is a string');
                } else if( typeof userInput === 'number') {
                    console.log("yep it's a number");
                } else if( typeof userInput === 'boolean') {
                    console.log("yep, its's a boolean");
                }
            }

Here I have created the function, "whatTypeIn" which checks the variable "userInput" to see what type of code it is with an if else statement. I can now recall the whole code to find the type with "whatTypeIn()".
